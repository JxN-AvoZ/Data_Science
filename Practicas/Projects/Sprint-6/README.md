# Análisis de ventas y éxito de videojuegos para la tienda online Ice


## Descripción del proyecto

En este proyecto se realiza un __análisis de datos para la tienda online Ice__, dedicada a la venta de videojuegos a nivel mundial. La empresa cuenta con información histórica sobre ventas, reseñas de usuarios y críticos, géneros, plataformas (como Xbox, PlayStation o PC) y clasificaciones por edad (ESRB). El objetivo es __identificar patrones que permitan determinar si un videojuego es exitoso o no__, con el fin de detectar proyectos prometedores y planificar campañas publicitarias más efectivas.

El análisis se sitúa en un contexto hipotético donde es __Diciembre de 2016__ y se desea utilizar los datos históricos disponibles para __planificar estrategias comerciales para el año 2017__. Más allá del año específico, el enfoque principal del proyecto es adquirir experiencia práctica en análisis de datos reales y en la toma de decisiones basada en evidencia.


## Objetivo

Identificar los factores que influyen en el éxito comercial de un videojuego, analizando ventas, plataformas, géneros, reseñas y clasificaciones por región, para apoyar la planificación de campañas publicitarias y decisiones estratégicas de negocio.


## Preparación de los datos

Durante la fase de preparación se realizan las siguientes tareas:

* Normalización de los nombres de las columnas.

* Conversión de tipos de datos a los formatos adecuados.

* Análisis y tratamiento de valores ausentes, incluyendo casos especiales como `TBD`.

* Justificación de las decisiones tomadas respecto a datos faltantes.

* Cálculo de las __ventas globales_ por juego a partir de las ventas regionales.

Esta etapa asegura que los datos sean consistentes y aptos para un análisis confiable.

### Análisis exploratorio

El análisis de datos se centra en:

* Evolución del número de videojuegos lanzados por año y relevancia de cada período.

* Comparación de ventas entre plataformas y estudio de su ciclo de vida.

* Identificación de plataformas líderes, emergentes y en declive.

* Evaluación de diferencias en ventas mediante diagramas de caja.

* Análisis de la relación entre reseñas (usuarios y críticos) y ventas.

* Comparación de ventas de los mismos juegos en distintas plataformas.

* Estudio de la distribución de ventas por género y detección de los géneros más rentables.

A partir de estos análisis se selecciona el período de datos más relevante para construir conclusiones aplicables al año 2017.

### Perfil de usuario por región

Se construyen perfiles específicos para cada región principal (__Norteamérica__, __Europa__ y __Japón__), analizando:

* Las cinco plataformas más populares en cada región.

* Los cinco géneros más vendidos y sus diferencias regionales.

* El impacto de las clasificaciones ESRB en las ventas según la región.

Este enfoque regional permite comprender mejor las preferencias del mercado global.

### Pruebas de hipótesis

El proyecto incluye pruebas estadísticas para validar afirmaciones clave, tales como:

* Comparación de las calificaciones promedio de usuarios entre Xbox One y PC.

* Comparación de las calificaciones promedio entre los géneros de Acción y Deportes.

Se definen hipótesis nulas y alternativas, se establece un nivel de significancia y se justifica el criterio estadístico utilizado, interpretando los resultados desde una perspectiva de negocio.


## Resultado esperado

Al finalizar el proyecto se obtiene:

* Un análisis integral del mercado de videojuegos basado en datos históricos.

* Identificación de plataformas y géneros con mayor potencial comercial.

* Conclusiones fundamentadas sobre el impacto de reseñas y clasificaciones en las ventas.

* Recomendaciones útiles para la planificación de campañas publicitarias.


## Conclusiones

Este proyecto simula un escenario real de análisis de datos en la industria del entretenimiento digital, integrando limpieza de datos, análisis exploratorio, segmentación regional y pruebas estadísticas. El resultado final proporciona una base sólida para comprender qué factores influyen en el éxito de un videojuego y cómo Ice puede utilizar esta información para tomar decisiones estratégicas informadas.
